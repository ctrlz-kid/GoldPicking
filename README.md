# GoldPicking

\>>C++ Exercise

#### 题目描述  
阿里巴巴走进了装满宝藏的藏宝洞。藏宝洞里面有 N 堆金币，第 i 堆金币的总重量和总价值分别是 m<sub>i</sub>, v<sub>i</sub> 。

阿里巴巴有一个承重量为 T 的背包，但并没办法将全部的金币都装进去。他想装走尽可能多价值的金币。所有金币都可以随意分割，分割完的金币重量价值比（也就是单位价格）不变。

请问阿里巴巴最多可以拿走多少价值的金币？  
#### 输入格式
输入共 2 行：  
  
第 1 行，两个整数 N、T；  
  
接下来 N 行，每行两个整数 m<sub>i</sub>, v<sub>i</sub>
